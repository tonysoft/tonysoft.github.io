You can set the vega-spec property specifying a URL which has a valid Vega Chart Specification. Note that this Sample Specification references Chart Data directly from within the Specification itself.

You can set the vega-spec-json property passing a Vega Chart Specification which you have already loaded. Note that this referenced Sample Specification DOES NOT include Data.

You can load Vega Chart Data independently from the Vega Chart Specification by using the vega-data JSON property. If you do this, the newly specified Data will override any Data previously loaded.

Check out a variety of Vega Chart Samples. When you check out an example like the Bar Chart, be sure to View in Online Vega Editor where you can experiment and modify the Sample to your own purposes.

The Vega-Lite Sample Gallery has even more samples. But, when viewing these samples in the View in Online Vega Editor be sure to ultimately use the Vega Specification (dropdown in the upper left) as the vega-componenbt uses the full Vega Vocabulary.
