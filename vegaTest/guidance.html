You can set the vega-spec property specifying a URL which has a valid Vega Chart Specification. Note that this Sample Specification references Chart Data directly from within the Specification itself.

You can set the vega-spec-json property passing a Vega Chart Specification which you have already loaded. Note that this Sample Specification DOES NOT include Data.

You can load Vega Chart Data independently from the Vega Chart Specification by using the vega-data JSON property. If you do this, the newly soecified Data will override any Data previously loaded.
